═══════════════════════════════════════════════════════════════════════════════
  METHOD 1: AUTOMATIC (RECOMMENDED)
═══════════════════════════════════════════════════════════════════════════════

1. Copy file to Windows:
   - Transfer 'windows_mcp_installer.ps1' to your Windows PC

2. Run PowerShell as User (not admin needed):
   - Right-click Windows Start → Windows PowerShell

3. Navigate to file location:
   cd Downloads

4. Run installer:
   .\windows_mcp_installer.ps1

5. Follow prompts (choose option 3 for both local + VPN)

6. Restart Claude Desktop

═══════════════════════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "Connection refused" or "Cannot reach API"
Solution: Test API first in PowerShell:

  Invoke-RestMethod -Uri "http://192.168.50.10:8080/api/v1/ping"

  Should return: {"status":"pong"}

Problem: "Node.js not found"
Solution: Install Node.js from https://nodejs.org

Problem: MCP not showing in Claude
Solution:
  1. Check config file exists at: %APPDATA%\Claude\claude_desktop_config.json
  2. Restart Claude Desktop (close completely first)
  3. Check Claude Desktop logs
